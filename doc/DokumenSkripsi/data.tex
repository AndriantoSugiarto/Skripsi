\abstrakINA{
	Program Studi Teknik Informatika dalam proses melakukan kurikulum 2018. Pada kurikulum 2018 terdapat perbedaan kode mata kuliah, nilai akhir, dan syarat kelulusan dengan kurikulum 2013. Oleh karena itu, perlu dilakukan perubahan pada SIAmodels.
	
	IFStudentPortal merupakan aplikasi berbasis web yang dibuat menggunakan Play Framework untuk Teknik Informatika UNPAR. Data yang diolah IFStudentPortal diambil langsung dari StudentPortal Baru dengan memanfaatkan ekstrasi data situs web yang diimplementasikan menggunakan \textit{jsoup}. Untuk mendapatkan data yang dibutuhkan, perlu dilakukan analisis perubahan StudentPortal terhadap pengambilan data. Data yang telah diambil menggunakan jsoup dapat diseleksi dengan menggunakan CSS \textit{Selector} kemudian data tersebut akan diolah ke dalam SIAModels.
	
	Pengujian dari aplikasi IFStudentPortal dilakukan oleh beberapa mahasiswa dengan kriteria yang beragam. Berdasarkan hasil pengujian, aplikasi dapat berjalan dengan baik dan memberikan hasil yang tepat pada seluruh mahasiswa kecuali mahasiswa angkatan 2018 yang berbeda format NPM-nya. Hasil pengujian aplikasi IFStudentPortal membuktikan bahwa ekstrasi data situs web bergantung dengan penyedia layanan data untuk memenuhi kebutuhkan akademik mahasiswa Program Studi Teknik Informatika UNPAR.
}
\abstrakENG{
The Informatics Engineering Program is in the process of carrying out the  curriculum 2018. In the curriculum 2018 there are differences in course codes, final grades, and graduation requirements with the curriculum 2013. Therefore, changes need to be made to SIAmodels.

IFStudentPortal is a web-based application created using Play Framework for UNPAR Informatics Engineering. Data processed by IFStudentPortal is taken directly from the New StudentPortal by utilizing web site data extraction implemented using \ textit {jsoup}. To get the data needed, it is necessary to analyze Student Port changes to data retrieval. Data that has been taken using jsoup can be selected using CSS \ textit {Selector} then the data will be processed into SIAModels.

Testing of the IFStudentPortal application was performed by some students with various criteria. Based on the results of the test, the application can run well and provide the right results for all students except students of class 2018 in different NPM formats. The results of testing the IFStudentPortal application prove that the extraction of website data depends on data service providers to fulfill the needs of student academics at the UNPAR's Informatics Engineering students.
}
